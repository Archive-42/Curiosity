## Get all domains

### Request

```
GET /domains
```

### Headers

| Name          | Example                          |
| :------------ | :------------------------------- |
| Authorization | `Authorization: Bearer :tokenId` |

### Response

```
Status: 200 OK
```

```json
{
  "type": "domains",
  "data": [
    {
      "type": "domain",
      "data": {
        "id": ":domainId",
        "title": "Example",
        "created": "1475491394341",
        "updated": "1475491394341"
      }
    }
  ]
}
```

## Add a domain

### Request

```
POST /domains
```

```json
{
  "title": "Example"
}
```

### Headers

| Name          | Example                          |
| :------------ | :------------------------------- |
| Authorization | `Authorization: Bearer :tokenId` |

### Parameters

| Name  | Type   | Required | Description          |
| :---- | :----- | :------- | :------------------- |
| title | String | true     | Title of the domain. |

### Response

```
Status: 201 Created
Location: /domains/:domainId
```

```json
{
  "type": "domain",
  "data": {
    "id": ":domainId",
    "title": "Example",
    "created": "1475491394341",
    "updated": "1475491394341"
  }
}
```
