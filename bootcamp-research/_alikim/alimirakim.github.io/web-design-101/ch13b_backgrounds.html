background-color

This background color property fills the canvas behind any element, which includes the content area, padding space, and to the outer edges of the border. It is transparent by default.

opacity

Though the color values themselves can be input to include alpha transparency, this property can be applied on its own to effect the opacity of an element.

Examples:

  • h4 {color: red; background: white; opacity: 1}"

  • h4 {color: red; background: white; opacity: 75%}"

  • h4 {color: red; background: white; opacity: 0.5}"

  • h4 {color: red; background: white; opacity: 25%}"

    background (shortcut property)

    The values for this property are all optional and can be placed in any order. The exception is when providing coordinates for the background-position property: the horizontal value must appear first, immediate followed by the vertical value. Be aware that if a value is omitted, it will be reset to the attribute's default value, not the browser's default style.

    Multiple Backgrounds

    When layering multiple background images, whichever image is listed last is at the bottom and the image listed first sits at the top, unlike the usual 'last one wins' rule for CSS items. Try squishing the window to see the three images overlap each other.

    The two examples yield the same result. They are just alternate options to accomplish the same thing.

    Examples:

    body {
    background-image: url(media/image_1.png), url(media/image_2.png), url(media/image_3.png);
    background-position: left top, center center, right bottom;
    background-repeat: no-repeat, no-repeat, no-repeat;
    }
    body {
    background:
      url(media/image_1.png) left top no-repeat;
      url(media/image_2.png) center center no-repeat;
      url(media/image_3.png) bottom right norepeat;
      }

    radial-gradient()

    You can specify shape, size, and center position of a radial gradient. Shape: circle, ellipse. Size: length units or percentage (width height). Position: center center by default. Keywords: closest-side, closest-corner, farthest-side, farthest-corner.

    • background-image: radial-gradient(circle, yellow, green)
    • background-image: radial-gradient(200px 80px, aqua, green)
    • background-image: radial-gradient(farthest-side at right bottom, yellow, orange 50%, purple)
